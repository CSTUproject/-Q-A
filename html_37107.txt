                    <div class="b_con" style="line-height:25px;" id="b_con"><strong>★问题1：理解规范时如何看待&ldquo;法无禁止皆可为&rdquo;与&ldquo;法无授权即禁止&rdquo;?</strong><br />
&nbsp; &nbsp; 提出本问题的原因：<br />
&nbsp; &nbsp; 答案请关注&ldquo;张耀泽老师宣贯&rdquo;。<br />
<br />
<strong>★问题2：图示突破规范条文时，是否有条文同等法律效力？如<a href="http://gf.1190119.cn/article-27798.htm">《图示》5.5.8</a>?</strong><br />
&nbsp; &nbsp; 提出本问题的原因：明确《规范》、《标准》、《图示》、《指南》等的效力和优先级<br />
&nbsp; &nbsp; 答案请关注&ldquo;张耀泽老师宣贯&rdquo;。<br />
&nbsp; &nbsp; 规范适用性（<a href="https://gf.1190119.com/article-27227.htm">《建规》1.0.2</a>）<br />
<br />
<strong>★问题3：&ldquo;等&rdquo;中是否补充&ldquo;汽车库&rdquo;、&ldquo;钢铁冶金企业&rdquo;；&ldquo;宜&rdquo;是否改为&ldquo;应&rdquo;？</strong><br />
&nbsp; &nbsp; 提出本问题的原因：&ldquo;等&rdquo;字的解读。<br />
&nbsp; &nbsp; 答案敬请期待后续版本完善。<br />
&nbsp; &nbsp; 防火分隔（<a href="https://gf.1190119.com/article-27227.htm">《建规》1.0.4</a>）<br />
<br />
<strong>★问题4：明晰本条的应用规则；明确建筑和场所的区别；理解同一使用功能建筑内的这些特殊的局部功能的安全疏散、设备设施等，按整体设计还是按局部设计？比如：1）电商网店同附设仓储之间的防火分隔要求？2）学校生活楼中，一层为厨房、二层为餐厅、三层以上为宿舍。宿舍部分的楼梯是否需要与底部的餐厅分开？</strong><br />
&nbsp; &nbsp; 提出本问题的原因：（《公津建字【2015】39号》：2.0h隔墙+乙级防火门）<br />
&nbsp; &nbsp;&nbsp;<a href="https://gf.1190119.com/article-27227.htm">《建规》1.0.4</a>条文说明：当同一建筑内，可能会存在<u>多种用途的房间或场所</u>，如办公建筑内设置的会议室、餐厅、锅炉房等，<u>属于同一使用功能。</u><br />
&nbsp; &nbsp; 1）<a href="https://gf.1190119.com/article-36795.htm">《浙江》3.4.12</a>：电商网店内附设临时仓储功能的，仓储部分与电子商务部分应采取耐火极限不低于2.0h的不燃烧体隔墙、乙级防火门进行防火分隔。<br />
&nbsp; &nbsp; 2）<a href="https://gf.1190119.com/article-37096.htm">《江苏》1.3.3.2</a>：学校生活楼中，三层以上宿舍的疏散楼梯间应独立设置。二层餐厅的疏散楼梯不应与宿舍楼梯间兼用。<br />
&nbsp; &nbsp; 规范适用性<a href="https://gf.1190119.com/article-27227.htm">《建规》（1.0.7）</a><br />
<br />
<strong>★问题5：<a href="https://gf.1190119.com/article-27227.htm">《建规》1.0.7</a>条、<a href="https://gf.1190119.com/article-4467.htm">《车火规》1.0.4</a>条中的&ldquo;除应符合本规范的规定外，尚应符合国家现行有关标准的规定&rdquo;，请问规范互参的范围及原则？ （比如汽车库只执行《车火规》）</strong><br />
&nbsp; &nbsp; 提出本问题的原因：<br />
&nbsp; &nbsp; 1、以汽车库为例，说明执行的关系。《车火规》是专业规范，《建规》是通用防火规范，按照专业规范优先的原则。设计与审查以《车火规》为主，而且，《车火规》与《建规》的要求抵触时，按《车火规》要求。汽车库消防电梯和防火卷帘等的设置《车火规》没有要求的，不要求按《建规》做。<br />
&nbsp; &nbsp; 2、以洁净厂房为例，洁净厂房没有专门的设计防火规范，其中，《<a href="https://gf.1190119.com/list-555.htm">洁净厂房设计规范</a>》中有关消防的要求是具体的、专业的，《建规》对厂房的要求仍然是通用的基本的总体的，因此，洁净厂房的防火设计，首先应符合《洁净厂房设计规范》的防火安全要求，对其要求执行《建规》的部分，应符合《建规》要求。<br />
&nbsp; &nbsp; 3、明哲保身的做法是，哪本规范要求严，执行哪本。<br />
<br />
<strong>★问题6：原消防建审部门传达的意见是&ldquo;改建扩建旧老建筑时可不按照&rdquo;宜&rdquo;的条款，新建建筑需要满足&ldquo;宜&rdquo;字相关条款&rdquo;。</strong><br />
&nbsp; &nbsp; 提出本问题的原因：&ldquo;宜&rdquo;字的解读<br />
&nbsp; &nbsp; 《乌鲁木齐》五.31：表示允许稍有选择，在条件许可时首先应这样做。<br />
<br />
<strong>★问题7：顿号造成错误解读</strong><br />
&nbsp; &nbsp; 提出本问题的原因：表达有歧义<br />
&nbsp; &nbsp; 答案敬请期待后续版本完善。<br />
<br />
<strong>★问题8：<a href="https://gf.1190119.com/article-27251.htm">《建规》5.5.17.4</a>，观众厅、&hellip;&hellip;、营业厅等，&ldquo;等&rdquo;字表示未穷举，其中是否包括：图书馆的大空间阅览室？高校实验楼的大空间型实验室？</strong><br />
&nbsp; &nbsp; 提出本问题的原因：&ldquo;等&rdquo;字的解读<br />
&nbsp; &nbsp; 《指南》P155：建筑的分类，应综合考虑建筑的高度、实际用途、发生火灾后可能产生的后果、扑救难易程度等，分类中的建筑称谓只表示其使用功能或用途，<u>实际设计中不应拘泥于建筑的称呼</u>，而应考虑上述主要因素.在表5.1.1中未明确的建筑，其分类可以根据上述原则釆用类比方法比照表5.1.1中的相应规定项来确定。<br />
&nbsp; &nbsp; 《指南》P210：歌舞娱乐放映游艺场所的范围<u>不应受上述场所称呼的限制</u>，而应根据具体场所的实际用途是否与规范规定的用途一致来确定其防火设计要求。<br />
&nbsp; &nbsp; 《福建》1.11：按照&ldquo;等内不等外&rdquo;的原则掌握，即仅指具体提到的建筑类型，不应延伸至其他类似建筑。<br />
&nbsp; &nbsp;&nbsp;<a href="https://gf.1190119.com/article-36493.htm">《广州》2.5.10</a>、《广东》2.5.19.18：列举内容（图书馆的大空间阅览室和高校实验楼的大空间型实验室）属于多功能厅，可参照&ldquo;其室内任一点至最近疏散门或安全出口的直线距离不应大于30m&rdquo;执行。<br />
<br />
<strong>★问题9：<a href="https://gf.1190119.com/article-27268.htm">《建规》第 8.3.1.2</a>&ldquo;占地面积大于1500㎡或总建筑面积大于3000㎡的单、多层制鞋、制衣、玩具及电子等类似生产的厂房&rdquo;，该条款中&ldquo;等类似生产的厂房&rdquo;需进一步明确。</strong><br />
&nbsp; &nbsp; 提出本问题的原因：&ldquo;等&rdquo;字的解读<br />
&nbsp; &nbsp;&nbsp;<a href="https://gf.1190119.com/article-37091.htm">《江苏》1.1.23</a>：规范要求设置自动喷水灭火系统的生产厂房或场所，可参照《指南》P391页表F8.7。
<div style="text-align: center;">

<br />
<strong>问题10：设置喷淋后，距离仍然不能扩大的情况？</strong><br />
&nbsp; &nbsp; 提出本问题的原因：<br />
&nbsp; &nbsp; 《指南》P249：规范规定的<u>所有涉及设置条件的数值</u>，如无明确规定，均不能根据规范有关调整要求进行改变。因此，一座全部设置自动灭火系统的建筑中位于疏散走道尽端，只设置个疏散门的房间，其房间内任一点至疏散门的直线距离仍不应大于15m，且建筑面积不应大于200㎡，疏散门的净宽度不应小于1.40m。<br />
<br />
